# موقع شخصي لطبيب (Doctor Portfolio Website)

موقع تعريفي شخصي لطبيب بتصميم عصري وحديث (Modern UI/UX)، تم بناؤه باستخدام HTML, CSS, JavaScript فقط بدون أي أطر عمل (Frameworks).

## 📂 هيكلية الملفات (Folder Structure)

```
doctor_portfolio/
├── index.html          # الصفحة الرئيسية (تحتوي على جميع الأقسام)
├── css/
│   └── modern.css      # ملف التنسيقات (تصميم عصري، Glassmorphism، Responsive)
├── js/
│   └── main.js         # ملف التفاعل (القائمة، التحريكات، النموذج)
├── assets/             # مجلد الصور (فارغ حالياً، مجهز لإضافة الصور)
└── README.md           # ملف التعليمات
```

## ✨ المميزات

- **تصميم متجاوب (Responsive)**: يعمل بامتياز على الهواتف، الأجهزة اللوحية، والحواسيب.
- **واجهة عصرية (Modern UI)**: استخدام ألوان طبية هادئة، ظلال ناعمة، وتأثيرات زجاجية (Glassmorphism).
- **تفاعلات سلسة**: تنقل ناعم بين الأقسام (Smooth Scrolling) وظهور تدريجي للمحتوى (Scroll Reveal).
- **جاهز للتطوير**: الكود منظم ومعلق عليه (Commented) ليسهل ربطه بـ Backend لاحقاً.
